EPA-Published Documents

While much of my current work is proprietary, I’m able to share some public-facing documents from my contract with the United States Environmental Protection Agency (EPA). Below is a selection of these texts.

Pre-Production Release Notes | Read Here

The purpose of these release notes are to inform the drinking water user community of the Drinking Water State-Tribal-Federal Information Exchange System (DW-SFTIES) development updates on a bi-monthly cadence. Each set of release notes contains the following sections: 1. What is included in the release, 2. New Features, 3. Enhancements, 4. Fixes, 5. Known Issues, 6. User Acceptance Testing Summary, 7. Goals for future releases, and 8. References.

Concept of Operations (CONOPS) | Read Here

This Concept of Operations (CONOPS) is both an analysis and a formal document that describes how the DW-SFTIES and associated components (applications, web services, etc.) will operate to achieve business, functional, and technical requirements.

Data Model Analysis Summary Report | Read Here

The purpose of this document is to summarize the analysis and decisions made during the evaluation phase for the Drinking Water State-Federal-Tribal Information Exchange System (DW-SFTIES) data model.

Ground Water Rule (GWR) Data Entry Instructions (DEIs) | Read Here

The Ground Water Rule (GWR) Data Entry Instructions (DEIs) document provides the most common GWR violations in the form of example scenarios. This DEI also provides guidance about data fields that need to be reported for each violation scenario.

2022 General Assistance Program (GAP) Guidance | Read Here

This Guidance provides information to Tribes and intertribal consortia on how to develop, apply for, and manage Indian Environmental General Assistance Program (GAP) financial assistance agreements, generally referred to as GAP grants. It also describes how the U.S. Environmental Protection Agency (EPA) administers GAP grants to ensure national consistency in EPA’s actions when negotiating, reviewing, and administering GAP awards.
